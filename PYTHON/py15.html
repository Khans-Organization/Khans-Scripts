Sets in Python

A set is an unordered collection of unique elements in Python. Unlike lists or tuples, sets do not allow duplicate values, and their elements are not stored in any particular order. Sets are mutable, meaning you can add or remove items after creation, but the elements themselves must be immutable (like strings, numbers, or tuples).

Key Characterstics:

  • Immutable: Once created, the elements in a tuple cannot be changed.
  • Unodered: Tuples maintain the order of elements, meaning that the order in which elements are added is preserved.
  • Can Contain Mixed Data Types: Tuples can hold elements of different data types (e.g., integers, strings, and even other tuples).
  • Indexable: Elements in a tuple can be accessed by their index, starting from 0.

Syntax:

Tuples are defined by enclosing elements within parentheses '()'.

# Creating a tuple
my_tuple = (1, 2, 3, "apple", "banana")

Accessing Values

You can access elements in a tuple using indexing and slicing.

Example:

# Accessing elements by index
print(my_tuple[0])  # Output: 1
print(my_tuple[3])  # Output: apple
            
# Slicing a tuple
print(my_tuple[1:4])  # Output: (2, 3, 'apple')

Packing and Unpacking Tuples:

  • Packing: Assigning multiple values to a single tuple variable.
  • Unpacking: Extracting values from a tuple into individual variables.

Example:

# Packing
my_tuple = 1, 2, "apple"
# Unpacking
x, y, z = my_tuple
print(x)  # Output: 1
print(y)  # Output: 2
print(z)  # Output: apple

Nested Tuples:

Tuples can also contain other tuples as elements, allowing for nested data structures.

nested_tuple = (1, 2, (3, 4), 5)
print(nested_tuple[2])  # Output: (3, 4)

Conclusion

Tuples in Python, being immutable, offer only a couple of built-in methods: count() to determine how many times an element appears, and index() to find the position of an element's first occurrence. Although limited, these methods, along with the in operator for checking membership, provide essential functionality for working with tuples effectively.
