Time module in Python

Python provides several built-in modules to work with time, dates, and timestamps. These functions are essential for managing tasks that need to be executed after a delay or repeatedly at regular intervals.

1. time.time() Function

The "time.time()" function is used to execute the current time in seconds since the Epoch (January 1, 1970, 00:00:00 UTC).

Syntax:

import time
current_time = time.time()
  • function: Tells the current time since Epoch.

Example:

import time

current_time = time.time()
print("Current Time:", current_time)
# Output: Current Time: 1693226225.4843676

2. time.sleep() Function

The "time.sleep()" function in Python is used to pause the execution of the program for a specified number of seconds. It's often used when you want to introduce a delay in your program's execution, for instance, to create a pause between actions.

Syntax:

time.sleep(seconds)
  • seconds: A non-negative number representing the number of seconds for which the program should be paused. This can be an integer or a floating-point number to specify fractions of a second.

Example:

print("Start")
time.sleep(3)  # Pauses for 3 seconds
print("End after 3 seconds")
# Output:
Start
(Program pauses for 3 seconds)
End after 3 seconds

In this example, the '3' is passed to time.sleep() function and the program is executes 1st line after 3 seconds last line is executed

3. time.ctime([secs])

"time.ctime([secs])": Converts time in seconds since the Epoch to a readable string format. If no seconds are provided, it uses the current time.

Example:

readable_time = time.ctime()
print("Readable Time:", readable_time)

output:

Readable Time: Mon Aug 28 12:17:05 2024

4. Practical Use Cases

  • "setTimeout()" is commonly used for delaying tasks, creating animations, or simulating wait times.

    • Example:

    function delayedGreeting() {
    console.log("This message appears after a delay.");
}

setTimeout(delayedGreeting, 1500);
  • "setInterval()" is useful for tasks that need to be repeated at regular intervals, such as updating a clock, polling a server, or creating a slideshow.

    • Example:

    let count = 0;
let counter = setInterval(function() {
    count++;
    console.log(count);
    if (count === 5) {
        clearInterval(counter); // Stops the interval after 5 counts
    }
}, 1000);

In this example, the "displayTime" function will be executed every second for 5 seconds, after which the interval will be cleared.
