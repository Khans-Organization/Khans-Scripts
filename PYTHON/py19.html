File handling

File handling in Python involves reading from and writing to files on your computer. Python provides built-in functions and methods that allow you to manipulate files easily.

Opening a file:

In Python, the 'open()' function is used to open a file.

Syntax:

file_object = open(filename, mode)

Explanation:

  • Mutable and Immutable
    • filename:
      • The name (and path) of the file to open.
    • mode:
      • A string that specifies the mode in which the file is opened. Common modes include:
      • 'r':
        • Read (default mode) - opens the file for reading.
      • 'w':
        • Write - opens the file for writing (creates a new file or truncates an existing file).
      • 'a':
        • Append - opens the file for writing, appending to the end of the file if it exists.
      • 'b':
        • Binary mode - opens the file in binary mode (e.g., 'rb' or 'wb').
      • '+':
        • Update mode - allows both reading and writing (e.g., 'r+', 'w+').
