Diamond Pattern

Unleash your creativity with Python! This concise Python script generates intricate diamond patterns effortlessly. By adjusting a single parameter, you can craft diamonds of varying sizes, making it perfect for decorative elements, educational exercises, or purely artistic pursuits. The script employs nested loops and string manipulation techniques to ensure each diamond maintains its symmetrical allure, resulting in visually captivating designs. Whether you're a beginner or an experienced coder, this Python code opens up a world of geometric possibilities, inviting you to explore and create mesmerizing diamond patterns with ease.

lines = int(input("Enter the number of lines: "))

for i in range(1, lines + 1):
      print(" " * (lines - i) + "*" * (2 * i - 1))
for j in range(lines - 1, 0, -1):
      print(" " * (lines - j) + "*" * (2 * j - 1))
